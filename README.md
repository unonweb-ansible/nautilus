## Install Fonts

- Author: Sahithyan K.
- Source: https://github.com/sahithyandev/nautilus-scripts

This script installs selected font files to the local user's font directory.
Usage

- Select the font files you want to install.
- Run the Install fonts script.
- The script will copy the fonts to ~/.local/share/fonts/.

Requirements
- None

## OCR (JPEG)

- Author: Sahithyan K.
- Source: https://github.com/sahithyandev/nautilus-scripts

This script performs Optical Character Recognition (OCR) on selected JPEG images and saves the extracted text to a file.
Usage

- Select the JPEG images you want to process.
- Run the OCR (Image) script.
- Enter the OCR language when prompted.
- The script will save the extracted text to a .txt file with the same name as the image.

Requirements
- Tesseract OCR

## Merge PDFs

- Author: Sahithyan K.
- Source: https://github.com/sahithyandev/nautilus-scripts

This script merges multiple PDF files into a single PDF.
Usage

- Select at least two PDF files.
- Run the Merge PDFs script.
- Choose the output file name and location.
- The script will merge the PDFs into the specified file.

Requirements
- pdfunite command
